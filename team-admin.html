Team Admin

This internal page helps update the Team page. Roles stay fixed, but names, start years, photos, favorite activities, and bios can change. GitHub Pages cannot save directly, so use the export buttons and downloaded renamed photos to apply updates. The Team data file belongs at assets/data/leaders.json.

After downloading, replace assets/data/leaders.json with the new leaders.json file, then commit and push.
