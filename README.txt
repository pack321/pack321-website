PACK 321 FAVICON PACKAGE

Files included:
- favicon.ico
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png
- android-chrome-192x192.png
- android-chrome-512x512.png
- site.webmanifest
- favicon-links.html

UPLOAD:
Place all files in the root/public folder of the website.

For a plain HTML website:
Paste the contents of favicon-links.html inside the <head> section.

For GitHub Pages:
Place the files beside index.html, or in the configured public root.

For Next.js:
Place the icon files and site.webmanifest inside /public.
